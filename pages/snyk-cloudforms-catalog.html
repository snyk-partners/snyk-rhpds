Secure your Containers with Snyk

Snyk empowers developers to easily find and fix vulnerabilities in applications running on OpenShift. This workshop guides users through using Snyk Container and Snyk Infrastructure as Code to identify and remediate security and configuration risks in a sample application deployed into OpenShift.

The RHPDS environment provides each user an OpenShift project with:

  • A deployment of Snyk's vulnerable Goof application
  • A fully deployed and running Snyk Controller
The steps below will cover the following topics:
  • Reviewing the security and configuration scan results in the Snyk UI, then
  • Finding and applying a more secure base image using Snykâ€™s upgrade guidance.
  • Securing the Goof application's deployment configuration within OpenShift
