Snyk Demo & Workshop

Secure your Containers with Snyk

Snyk empowers developers to easily find and fix vulnerabilities in applications running on OpenShift. This workshop guides users through using Snyk Container and Snyk Infrastructure as Code to identify and remediate security and configuration risks in a sample application deployed into OpenShift.

Workshop

This workshop is conducted using the Red Hat Product Demo System (RHPDS). Within RHPDS, the catalog item is in the Workshops category and is called Snyk for OpenShift. The RHPDS environment provides each user an OpenShift project with:

  • A deployment of Snyk's vulnerable Goof application
  • A fully deployed and running Snyk Controller
All projects will be on a shared cluster deployed specifically for this workshop. In RHPDS, before ordering, the catalog item description links to instructions and other details about performing the workshop.

Source

The source for the workshop can be found in GitHub.

A non-RHPDS version of this workshop, and other Snyk demos for Red Hat, can be found in the Snyk Academy.
